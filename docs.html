Getting started

Linking in the HTML File:

<link href="https://acoustica-ui.netlify.app/styles/index.css" rel="stylesheet" />

Importing in the CSS File:

@import url("https://acoustica-ui.netlify.app/styles/index.css" rel="stylesheet");

Alert

Alerts are used to describe a state that affects a system, feature, or page.

error_outline Danger! Indicates the dangerous / potentially negative action.
check_circle Success! This alert indicates the successful or positive actions.
info Info! This alert indicates neutral informative change or action.
warning_amber Warning! This alert indicates a warning which needs attention. 
<div class="alert alert-danger">
    <span class="material-icons-round alert-icon"> error_outline </span>
    Danger! Indicates a dangerous or potentially negative action.
</div>
        
<div class="alert alert-success">
    <span class="material-icons-outlined alert-icon"> check_circle </span>
    Success! Indicates a successful or positive action.
</div>
        
<div class="alert alert-info">
    <span class="material-icons-outlined alert-icon"> info </span>
    Info! Indicates a neutral informative change or action.
</div>
        
<div class="alert alert-warning">
    <span class="material-icons-round alert-icon"> warning_amber </span>
    Warning! Indicates a warning that might need attention.
</div>

Buttons

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

<button class="btn btn-simple"> Simple Button </button>

<button class="btn btn-outline"> Outline Button </button>

<button class="btn btn-clear"> Clear Button </button>

    <span class="material-icons md-18"> add_shopping_cart </span>
